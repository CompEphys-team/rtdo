# rtdo
Real-time dynamic observer
Before compiling and running, ensure the following:
* Clone with `git clone --recurse-submodules`, or run `git clone submodule update --recursive` after cloning to clone into submodules
* Non-standard dependencies:
  * GNU scientific library (GSL), usually packaged as libgsl-dev
  * Comedilib, usually packaged as libcomedi-dev
  * CUDA toolkit > 9
